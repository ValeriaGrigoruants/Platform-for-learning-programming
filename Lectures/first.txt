<p style=" margin-top:0px; margin-bottom:0px; margin-left:0px; margin-right:0px; -qt-block-indent:0; text-indent:0px;">An array is a series of elements of the same type placed in contiguous memory locations that can be individually referenced by adding an index to a unique identifier.<br /><br />That means that, for example, five values of type <span style=" font-family:'Courier New';">int</span> can be declared as an array without having to declare 5 different variables (each with its own identifier). Instead, using an array, the five <span style=" font-family:'Courier New';">int</span> values are stored in contiguous memory locations, and all five can be accessed using the same identifier, with the proper index.<br /><br />For example, an array containing 5 integer values of type <span style=" font-family:'Courier New';">int</span> called <span style=" font-family:'Courier New';">foo</span> could be represented as:<br /><br /><img src="C:\\Users\\Valeria\\Documents\\LearnProgramming\\arrays1.png" /> <br />where each blank panel represents an element of the array. In this case, these are values of type <span style=" font-family:'Courier New';">int</span>. These elements are numbered from 0 to 4, being 0 the first and 4 the last; In C++, the first element in an array is always numbered with a zero (not a one), no matter its length.<br /><br />Like a regular variable, an array must be declared before it is used. A typical declaration for an array in C++ is:<br /><br /><span style=" font-family:'Courier New';">type name [elements];<br /></span><br />where <span style=" font-family:'Courier New';">type</span> is a valid type (such as <span style=" font-family:'Courier New';">int</span>, <span style=" font-family:'Courier New';">float</span>...), <span style=" font-family:'Courier New';">name</span> is a valid identifier and the <span style=" font-family:'Courier New';">elements</span> field (which is always enclosed in square brackets <span style=" font-family:'Courier New';">[]</span>), specifies the length of the array in terms of the number of elements.<br /><br />Therefore, the <span style=" font-family:'Courier New';">foo</span> array, with five elements of type <span style=" font-family:'Courier New';">int</span>, can be declared as:<br /><br /></p>
